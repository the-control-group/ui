# Notification
### Floating and optionally dismissible notification component

## Generic:
Use `Notification` with no other props as a dismissible notification that drops down from the top of the screen.

<Playground style={{position: 'relative', height: 75}}>
	<Notification style={{position: 'absolute'}} onDismiss={() => alert('Notification Dismissed')}>
		This is a Notification. Click the X to dismiss.
	</Notification>
</Playground>

## Timer:
Use the `timer` prop to create a notification that dismisses itself after a set amount of seconds.

<Playground style={{position: 'relative', height: 75}}>
	<Notification timer="5" style={{position: 'absolute'}} onDismiss={() => console.log('Notification Timed Out')}>
		This is a timed notification. It will dismiss itself after 5 seconds.
	</Notification>
</Playground>
