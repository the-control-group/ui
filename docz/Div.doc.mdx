# Div
### Container styles and properties.

## Generic:
Use `Div` with no other props as a replacement for `div` in order to expose common UI properties to your containers.

<Playground>
	<Div>This is a Div</Div>
</Playground>

## Items:
Use the `item` prop to create a distinct item within your container.

<Playground>
	<Div item>Div item #1</Div>
	<Div item>Div item #2</Div>
	<Div item>Div item #3</Div>
</Playground>

## Card:
Use the `card` prop to create a 3D card container.

<div className="card-example"></div>

<Playground>
	<Div card>This is a Div card</Div>
</Playground>

## Card Subsection:
Use the `subsection` prop on elements under a `card` to create distinct sections within your card.

<div className="card-example"></div>

<Playground>
	<Div card>
		<Div subsection>This is a Div subsection</Div>
		<Div subsection>This is a Div subsection</Div>
		<Div subsection>This is a Div subsection</Div>
	</Div>
</Playground>
